Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Each element can be nested inside other elemets, inorder to make a complex document structure.
  • All element have some universal atribute, and some specific to that element. The atribute can be used to modifiy what a element tag can do or how it looks.

CSS

  • Cascading style sheets or CSS for short, is used to style a html document giving it a particular look.
  • A CSS file must first be linked to an html document using <link>.
  • <link> has two primary atributes.
    • The 'rel' atribute specifies the relation to the html, in the case of a CSS file, its stylesheet.
    • The 'href' atribute specifies the location of the file, it can either be a url or a file path on the local computer.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

  • git init: initalize a git repository on the computer
  • git status: used to check to if the repository has initalized, it also looked at what branch that is active along with how far or behind is that branch with the main remote branch
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is used to make the webpage interactable.
  • JavaScript is a interpertated scripting lanague.
  • The <script> tag is where JavaScript code is put.
  • Like CSS, JavaScript can be linked to a HTML document using an external file using the <script> tag and a 'src' atribute which points to the location of a file.
